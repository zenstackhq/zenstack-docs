# Pinia Colada

[Pinia Colada](https://pinia-colada.esm.dev/) is a popular state management library for Vue.js applications. Thanks to a community integration by [Eugen Istoc](https://github.com/genu), you can seamlessly integrate Pinia Colada with ZenStack's automatic CRUD services.

For more details and documentation, visit the [GitHub repository](https://github.com/genu/zenstack-pinia-colada).
